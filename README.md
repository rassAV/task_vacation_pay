Задание - оконное приложение, которое будет хранить данные в БД и проводить какую-то обработку этих данных<br>
Тема - расчёт отпускных<br>

Формула расчёта отпускных:<br>
(зарплата / 29.3) * дней отпуска<br>
29.3 - среднее число дней в месяце<br>

Функционал:<br>
Добавление записи (авторасчёт отпускных)<br>
Вывод записей (всех или с фильтрацией по имени)<br>
Сортировка записей по возрастанию/убыванию для столбцов зарплата, дней отпуска, отпускные<br>
