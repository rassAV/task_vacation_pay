# task_vacation_pay

Задание - оконное приложение, которое будет хранить данные в БД и проводить какую-то обработку этих данных

Тема - расчёт отпускных



Формула расчёта отпускных:

(зарплата / 29.3) * дней отпуска

29.3 - среднее число дней в месяце



Функционал:

Добавление записи (авторасчёт отпускных)

Вывод записей (всех или с фильтрацией по имени)

Сортировка записей по возрастанию/убыванию для столбцов зарплата, дней отпуска, отпускные
